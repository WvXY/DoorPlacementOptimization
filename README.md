# MasterProject: Door Placement Optimization

## Dependencies
- PythonCDT : for generating constrained delaunay triangulation mesh

## Installation
### PIP
```bash
pip install numpy matplotlib PythonCDT
```
 
### Conda Environment
```bash
conda install numpy matplotlib 
pip install PythonCDT
```
